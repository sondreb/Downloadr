Flickr Downloadr

Release v

Flickr Downloadr is an app that enables you to find and pick photos to download from the Flickr.com service.

Focus for the app have always been on simplicity and elegance. The main features needed, in the simplest form. The software and service is hosted and provided by brain.no.

Search and Download

Search, select photos, pick folder, download and wait. You can search multiple times and pick photos from different searches, before you start the download process.

Type

It's possible to either search free text on all photos, or by selecting the "User" option in the dropdown to search for a specific user. A user can be found either with e-mail address or username.

Sizes

On the folder selection screen, you can also decide what size you want to download. The options are as followed:
Original - Any size
Large - 1024x768 pixels
Medium - 640x480 pixels
Small - 320x240 pixels
Square - 150x150 pixels

If you choose a download size larger than what is available, then you will get the closest photo available.

Files and Folders

As files are stored to your local machine, the file names is a combination of the unique photo id, with the size (e.g. "_m" and "_o") and then with license appended before the file extension (e.g. "_by" and "by-sa").


Existing files?

If there is existing files with the same filename, Flickr Downloadr will append a random four character value to the filename. This ensures no interuption during download.

User Profiles

On your personal profile or on other users's profiles, you will be able to see the photostream, albums, favorites and galleries.


The user's photostream can be filtered with license and you can search by title.


The albums will all be shown with a thumbnail preview for the user. When you select albums, the downloading will only be allowed for some of the photos. It might happen that you try to download albums with only restricted photos. During the download process, you will be informed about which photos that could not be downloaded. New folder is created for each gallery.


The favorites will returns all photos of all licenses, including those that are unavailable for download. Those photos will be marked with a circle X, indicating not available for download.


Galleries are collection of up-to 50 individual photos that one user have selected from his/her own photos, or someone else. Downloading galleries will only save those photos you are licensed to save. New folder is created for each gallery.

Shortcuts

ALT+ENTER: Toggle fullscreen mode.

Photo Licenses

You are allowed to search and view photos with any licenses, but only those a user allows you to or are allowed to your authenticated user (like all your own photos) is allowed to be downloaded to your local computer.


The default selection will only show photos that can be downloaded, but in the bottom options you can enable showing search results of photos that are not downloadable. You can click on the "..." button on those photos, to take you directly to the Flickr page where you can send a message to the user, asking them to change permissions.


This is useful as many using Flickr is doing it with the intention of sharing memories, but doesn't necessary understand all the licensing issues.
